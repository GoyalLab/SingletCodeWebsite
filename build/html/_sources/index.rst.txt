Welcome to the documentation of Singlet Code!
=======================================

Singlet code :ref:'<https://www.biorxiv.org/content/10.1101/2023.08.04.552078v2>'_ uses cell barcoding technology to identify **real singlets** in scRNAseq data. compares to existing doublet detection method and provides a *classifier* to detect singlets in *unbarcoded* datasets as well.
